# AI-Voice-Drawing

AI-Voice-Drawing 是一个语音优先的可编辑矢量绘图工具。
用户可以用语音或文本描述想法，LLM 会把自然语言编译成结构化绘图命令，Canvas 再逐步执行这些命令。生成结果不是一次性的整图，而是一组可继续编辑的独立对象场景。

视频演示demo：https://www.bilibili.com/video/BV1E6JP6cESY/?vd_source=5c638aa4012a07ef7ff64c926928a548

## 项目定位

和“一句话直接生一张图”的方案不同，AI-Voice-Drawing 的核心是：

- 先生成结构化图元命令，而不是直接生成位图
- 画布中的每个图形都是独立对象，可以继续修改
- 用户可以继续通过语音或属性面板编辑已有对象
- 同一轮命令按步骤执行，过程可视化、可撤销、可重做

## 技术流程

1. 语音或文本输入进入编辑器
2. 本地先处理 UI 命令，例如打开属性、查看某个对象属性
3. 普通绘图或编辑意图交给 LLM 编译为 JSON 命令数组
4. Canvas 执行器按 0–100 scene-space 逐步执行命令
5. 对象进入场景状态，可继续被选择、修改、删除、整体变换
6. 当前场景可导出为 PNG 或 SVG

## 当前主要功能

- 语音识别、实时转写、文本输入兜底
- 长语音草稿模式
- 自然语言编译为结构化绘图命令
- 逐步执行可视化与解析预览
- 对象级编辑：颜色、位置、尺寸、删除
- 组合对象分组、整体删除、整体移动、整体缩放
- Canvas 点击选择对象
- 中文属性面板与基础专业色板
- 语音控制右侧面板
- 语音查看对象属性
- 坐标网格参考层
- 撤销、重做、清空
- 导出 PNG
- 导出 SVG

## 支持的图元与命令

当前场景对象支持：

- `circle`
- `rect`
- `line`
- `text`

当前命令支持：

- `circle`
- `rect`
- `line`
- `text`
- `update`
- `delete`
- `clear`

可以直接尝试这些指令：

- `画一座带红色屋顶、两扇窗户和一扇门的小房子`

## 语音优先交互

项目支持两种语音模式：

- 即时模式：说完后按现有触发词自动提交
- 长语音模式：持续累积草稿，直到用户说“好了 / ok / 执行 / 开始执行 / 生成吧”才提交当前草稿

在长语音模式下，说出以下词语会停止持续监听：

- `彻底结束`
- `结束长语音`
- `停止长语音`
- `关闭长语音`

部分界面命令优先在本地处理，不调用 LLM，例如：

- `打开属性`
- `打开图层`
- `打开指令预览`
- `打开历史记录`
- `查看太阳属性`
- `查看房子门的属性`

## 导出

### PNG

- 导出当前 Canvas 看到的画面
- 当前版本优先导出所见内容，包含当前画布中的参考网格与选中高亮

### SVG

- 根据当前 scene objects 生成 SVG
- `viewBox` 固定为 `0 0 100 100`
- 支持 `circle / rect / line / text`
- 支持 `fill / stroke / strokeWidth / opacity`
- 文本内容支持普通文字与 `★`
- 当前不导出坐标网格层

## 本地运行

```bash
npm install
npm run dev
```

构建：

```bash
npm run build
```

如果需要语音识别，建议使用支持 Web Speech API 的浏览器，例如较新的 Chrome 或 Edge。

## 项目结构概览

- `src/App.jsx`
  编辑器主状态、命令编排、历史栈、导出与界面协作
- `src/llm/compiler.js`
  自然语言到结构化绘图命令的编译提示词与解析
- `src/commands/executor.js`
  命令执行器
- `src/commands/renderer.js`
  Canvas 渲染器
- `src/components/DrawCanvas.jsx`
  画布、自适应缩放、网格与对象选择
- `src/components/PropertyPanel.jsx`
  属性编辑面板
- `src/utils/exportScene.js`
  SVG 序列化与下载工具

## 当前限制

- 不支持复杂路径与自由曲线
- 不支持真正的 polygon / star 图元
- 五角星等符号当前通过 `text` 对象表现
- 不支持专业图层排序
- 不支持鼠标拖拽、框选、多选、旋转
- 不支持复杂 HSL / HSV 相对调色
- Web Speech API 依赖浏览器支持与麦克风权限
