# AI-Voice-Drawing | AI 语音绘图工具

用自然语言说出想法,AI 将其编译为可编辑的矢量场景,在画布上逐步绘制。

不只是"语音生图"——每个图形都是可单独修改的对象,支持对象级语音编辑("把太阳调暗一点")、逐步绘制可视化、撤销重做与风格切换。

## 计划功能

- 🎤 语音识别实时上屏
- 🧩 自然语言 → 结构化绘图命令(基础图形 / 复杂指令拆解 / 场景编译)
- 🎬 每个绘图步骤逐条可视化执行
- ↩️ 撤销 / 重做
- ✏️ 对象级语音局部修改
- 💬 模糊意图主动追问(精准模式)
- 🎨 风格切换(扁平 / 手绘)

详细的指令清单与设计方案见 [DESIGN.md](./DESIGN.md)(随开发持续更新)。

## 技术栈

React + Vite · Web Speech API · LLM 场景编译 · Canvas 2D 渲染

## 组合对象

支持为复合场景元素添加轻量分组元数据，并通过自然语言整体删除组合对象。组合对象整体移动和缩放由执行器基于组级 transform 确定性执行，LLM 只负责生成变换意图；缩放中心使用 scene-space 包围盒中心。这不是完整图层系统，也不是像素级精确变换。

## 精准编辑

支持点击 Canvas 中的单个对象进行选择，并在右侧属性面板中精确调整对象参数。坐标体系继续使用 0–100 scene-space，不依赖像素尺寸。

当前属性面板支持调整基础坐标、尺寸、填充色、描边色、透明度和描边宽度；自然语言编辑也支持基础的精确参数修改，例如移动到指定 x/y、设置尺寸、使用 HEX 颜色、设置 opacity 或 strokeWidth。

当前尚未实现鼠标拖拽、框选、多选、组合对象选择和 HSL / HSV 等复杂调色能力。

## 运行

```bash
npm install
npm run dev
```

## 第三方依赖声明

React / Vite(框架与构建)、LLM API(命令解析)、Canvas 2D API(画布渲染)。
场景编译 prompt、场景图数据结构、命令执行器、步骤可视化与撤销机制为本项目原创。
