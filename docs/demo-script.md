# AI-Voice-Drawing Demo Script

这份脚本面向录屏演示，按顺序操作即可。

## 演示目标

在 2–4 分钟内展示：

- 语音优先输入
- 结构化逐步绘制
- 对象级编辑
- 组合对象编辑
- 语音 UI 控制
- PNG / SVG 导出

## 演示前准备

- 启动项目：`npm run dev`
- 确认浏览器允许麦克风权限
- 打开编辑器首页
- 保持右侧默认显示“指令预览”

## 推荐录屏流程

### 1. 开场介绍

口播要点：

- 这是一个语音优先的可编辑矢量绘图工具
- 它不是一次性语音生图，而是把自然语言编译成可编辑对象

镜头：

- 展示三栏编辑器布局
- 稍微停留在中央 Canvas、右侧面板和底部输入栏

### 2. 生成组合对象

输入或说出：

`画一座带红色屋顶、两扇窗户和一扇门的小房子`

展示要点：

- 右侧出现解析步骤
- 画布逐步生成多个对象
- 房子的墙、窗户、门是独立对象

### 3. 组合对象整体变换

输入：

`把房子整体向右移动 5 个单位`

展示要点：

- 只移动房子，不影响其他对象
- 预览区能看出这是组合更新

### 4. 语音查看对象属性

输入：

`查看房子门的属性`

展示要点：

- 右侧自动切换到属性面板
- 门对象被选中
- 属性面板显示中文字段

### 5. 属性编辑

在属性面板中把门改成深棕色，并把透明度调到 `0.85`

也可以直接输入：

`把门改成深棕色，透明度设为 0.85`

展示要点：

- 编辑只影响目标对象
- 其他对象不被重画

### 6. 新增细节不覆盖旧对象

输入：

`画一个绿色太阳`

接着输入：

`在绿色太阳中间加一个五角星`

展示要点：

- 太阳保持绿色
- 五角星作为新对象加入
- 没有覆盖原太阳对象

### 7. 常识图形模板

输入：

`生成五星红旗`

展示要点：

- 红旗由红色背景和五颗黄色星星组成
- 星星通过 `text` 图元 `★` 表现

### 8. 本地 UI 命令

依次输入：

- `打开属性`
- `打开图层`
- `打开指令预览`
- `打开历史记录`

展示要点：

- 面板切换不需要调用 LLM
- 画布内容不变

### 9. 导出

点击：

- `导出 PNG`
- `导出 SVG`

展示要点：

- 能正常下载
- SVG 可保留矢量对象结构

### 10. 收尾

口播要点：

- AI-Voice-Drawing 把语音生成与可编辑对象场景结合起来
- 后续会继续完善图层、精细编辑和更强的导出能力

## 备用演示指令

- `画一棵树和一朵云`
- `把树整体放大一点`
- `查看左窗户属性`
- `把太阳移动到 x=80，y=20`
- `删除房子`
- `撤销`
- `重做`

## 演示注意事项

- 如果语音环境不稳定，可以全程使用文本输入
- 如果模型响应较慢，重点强调“结构化命令 + 对象可编辑”这一核心差异
- 导出按钮建议放在演示末尾，避免打断主流程
